# Gestión del Dashboard

El Dashboard de Administración permite gestionar los recursos clave del sistema de monitoreo.

## Gestión de Nodos

Ubicación: `/dashboard/nodes`

En esta sección, los administradores pueden:

1.  **Listar Nodos**: Ver todos los nodos registrados en el sistema.
2.  **Crear Nodos**: Registrar nuevos puntos de monitoreo especificando:
    - ID del Nodo (único)
    - Descripción
    - Coordenadas (Latitud/Longitud)
3.  **Editar Nodos**: Modificar la ubicación o descripción.
4.  **Eliminar Nodos**: Dar de baja un nodo (requiere confirmación).

> **Nota**: Los IDs de nodos son inmutables una vez creados.

## Gestión de Datos (Lecturas)

Ubicación: `/dashboard/data`

Permite visualizar y corregir los datos recolectados por los sensores.

### Funcionalidades
- **Filtrado**: Buscar lecturas por Nodo y rango de Fechas.
- **Edición**: Corregir valores erróneos de sensores (pH, Batería, etc.).
    - El sistema valida que los valores estén dentro de rangos físicos lógicos (ej. pH 0-14).
- **Eliminación**: Borrar registros de lecturas inválidos.

### Validación de Datos
El sistema implementa validación estricta (Zod) para asegurar la integridad de los datos:
- **pH**: 0 - 14
- **Batería**: 0% - 100%
- **Temperatura**: -50°C - 100°C
